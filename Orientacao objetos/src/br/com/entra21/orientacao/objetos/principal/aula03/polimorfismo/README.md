# Aprendendo polimorfismo

##Polimorfismo significa várias formas

> várias formas de exceutar o comportamento herdado

> - É necessário herança para  fazer as 3 formas básicas de polimorfismo

> 1. Apenas aproveitar os comportamentos herdados
> 2. Fazer completamente minha maneira escrevendo um metodo identico da herança
> 3. Sobreescrever com @Override o método herdado
	- Essa forma obriga o meu método sobreescrito ter a mesma sintaxe da herança
	- Posso aproveitaar o comportamento herdado com o Super e fazer um diferencial
	- Posso não aproveitar nada e fazer do meu jeito
