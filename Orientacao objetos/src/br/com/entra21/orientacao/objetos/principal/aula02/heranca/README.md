# Aprendendo sobre Herança

## Herança é a abstração para aproveitar atributos e métodos de uma outra classe

> Essa classe que esta provendo atributos e metodos podemos chamá-la de super classe
> Para realizar essa herança em uma sub classe temos que usar a palavra reservado  __extends__
> Quando uma sub classe herda atributos e metodos da super classe ão é mais necessário manter meus atributos e métodos escritos na minha sub classe
